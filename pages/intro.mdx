Bem-vindo à documentação da API Cpay.

Aqui você encontrará todos os recursos necessários para realizar integrações de forma simples, eficiente e segura.
## <Icon icon="Share" type="solid"/>  Quem somos?

A Cpay é um gateway de pagamento criado a partir da nossa própria necessidade de simplificar cobranças.
Percebemos que as soluções disponíveis no mercado eram excessivamente complexas, apresentando desafios como:
- Documentações extensas e pouco objetivas
- Várias maneiras diferentes de executar a mesma operação
- E até exigência de treinamentos para conseguir integrar

Com a Cpay, buscamos eliminar essas barreiras e entregar uma experiência de integração simples, rápida e eficiente.

Nossa solução? Uma plataforma que transforma a complexidade das Fintechs em uma API simples e intuitiva. Veja como é fácil:

```php
<?php

use Cpay\Client\Client;

$payment = (new Client($_ENV('CPAY_API')))->charge()->create(
    amount: 200,
    name: 'Brendo Oliveira', // OPCIONAL
    document: '123.456.789-01' // OPCIONAL
);

```

## <Icon icon="laptop" type="solid"/>  API descomplicada

Nossa API foi construída com dois princípios fundamentais:

1. **Idempotente**: Execute a mesma requisição quantas vezes precisar, sem efeitos colaterais
   ```php
   // Seguro para executar múltiplas vezes
   $payment->charge()->create(...);
   ```

2. **Consistente**: Nossa API retorna sempre uma opção padronizada, que pode conter dados de erro ou o resultado esperado.
Esse formato garante consistência em todas as operações e facilita a implementação em linguagens de tipagem forte, reduzindo ambiguidades e simplificando o tratamento de respostas.
