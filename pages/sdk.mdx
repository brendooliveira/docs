## <Icon icon="code" type="solid"/> O que são os SDKs da Cpay?

Os **SDKs (Software Development Kits)** da Cpay são bibliotecas que encapsulam a comunicação com nossa API.
Eles fornecem funções pré-construídas, tipagem forte (quando disponível) e tratamento de erros nativo, permitindo que você foque no que importa: integrar rapidamente os pagamentos ao seu produto.

<Card title="Por que usar nossos SDKs?" icon="star" horizontal>
  * **Integração simplificada** → Funções prontas para os principais endpoints da API  
  * **Tipagem forte** → Interfaces completas em linguagens com suporte a tipos  
  * **Tratamento de erros** → Gerenciamento automático de erros comuns  
  * **Menor curva de aprendizado** → Não é necessário conhecer todos os detalhes internos da API  
  * **Atualizações frequentes** → SDKs sempre alinhados com as últimas funcionalidades da plataforma  
</Card>

## <Icon icon="terminal" type="solid"/> Exemplos de uso

Veja como é simples criar uma cobrança usando o SDK oficial em PHP:

<CodeGroup>
```php PHP
// Instalação: composer require cpay/cpay-sdk-php
<?php
require_once 'vendor/autoload.php';

use Cpay\Client\Client;

// Inicialize o cliente com seu token de autenticação
$client = new Client('seu_token_aqui');

$valor = 2000; // R$ 20,00
$nome = 'João da Silva';
$documento = '123.456.789-00'; // CPF ou CNPJ

// Parâmetros opcionais
$opcoes = [
    'name' => 'Brendo Oliveira', //OPCIONAL
    'document' => '123.456.789-00', //OPCIONAL
    'expiration' => 3600, //OPCIONAL
    'email' => 'brendo@cpay.app.br', //OPCIONAL
    'phone' => '(15) 98765-4321', //OPCIONAL
    'comment' => 'Pagamento de serviço', //OPCIONAL
    'externalID' => 'pedido-123' // ID externo único //OPCIONAL
];

// Criar a cobrança
$cobranca = $client->charge()->create($valor, $opcoes);

// O ID da cobrança estará disponível em $cobranca['data']['id']
echo "Cobrança criada: " . $cobranca['data']['id'];

```
</CodeGroup>

## <Icon icon="question-circle" type="solid"/> Perguntas frequentes

<Accordion title="Como instalar o SDK da minha linguagem?">
  Cada SDK possui instruções de instalação no respectivo repositório do GitHub.  
  Em geral, você pode instalar usando o gerenciador de pacotes da linguagem (Composer, npm, pip, bundle, etc.).
</Accordion>

<Accordion title="O que fazer se encontrar um bug em um SDK?">
  Abra uma issue no repositório do SDK correspondente no GitHub.  
  Nossa equipe acompanha ativamente para corrigir bugs e melhorar os pacotes.
</Accordion>

<Accordion title="Ainda não há SDK para a linguagem que uso. E agora?">
  Fale conosco em <Icon icon="envelope" type="solid"/> contato@cpay.app.br informando a linguagem desejada.  
  Também incentivamos contribuições da comunidade!
</Accordion>
