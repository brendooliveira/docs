## Buscar um Saque por ID

Este endpoint permite que você recupere os detalhes de um saque específico usando o ID do saque.

<RequestExample>
  ```bash
  curl https://api.aobapay.com/v1/withdraw/wdl_xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx \
  -H "Authorization: Bearer <token>"
  ```
</RequestExample>
