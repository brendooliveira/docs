## O que é

- BaaS permite criar contas para seus clientes dentro do ecossistema AobaPay.
- Você define e recebe taxas sobre transações usando o endpoint de transferência interna.
- Ideal para modelos com repasse, split e controle de saldo por cliente/grupo.

## Como funciona

- Criação de conta: envia dados cadastrais e documentos para abrir a conta.
- Transferência interna: movimenta valores entre chaves Pix do seu grupo.
- Precificação: aplique sua taxa e calcule seu lucro sobre cada transação.
- Automação: integre via API e orquestre fluxos financeiros no seu produto.

## Para quem é

- Fintechs e startups que querem oferecer Pix sem operar como banco.
- Marketplaces que precisam receber e distribuir pagamentos via Pix.
- Plataformas digitais (apps, SaaS, marketplaces) com alto volume de transações.
- Empresas que querem embedded finance (Pix integrado ao próprio produto).
- Adquirentes e subadquirentes que desejam Pix via API ou white label.
- Empresas com pagamentos recorrentes (fornecedores, parceiros, reembolsos).
- Negócios que precisam de automação financeira e integração via API.
- Empresas buscando reduzir custos versus meios tradicionais de pagamento.
- Gateways com necessidade de Pix performático e personalizável.

## Benefícios

- Controle total: contas separadas por cliente, membros e chaves Pix.
- Receita própria: taxas configuráveis e lucro por transação.
- Escalabilidade: API performática para grande volume de operações.
- Conciliação simples: dados consistentes para auditoria e repasses.
